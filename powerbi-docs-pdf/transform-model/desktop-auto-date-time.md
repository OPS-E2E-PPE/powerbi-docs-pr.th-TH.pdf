## <a name="configure-auto-datetime-option"></a><span data-ttu-id="94caa-170">กำหนดค่าตัวเลือกวันที่/เวลาอัตโนมัติ</span><span class="sxs-lookup"><span data-stu-id="94caa-170">Configure auto date/time option</span></span>

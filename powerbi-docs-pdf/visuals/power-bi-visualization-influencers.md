<span data-ttu-id="53aa1-464">**คุณจะคำนวณผู้มีอิทธิพลหลักสำหรับการวิเคราะห์เชิงตัวเลขได้อย่างไร**</span><span class="sxs-lookup"><span data-stu-id="53aa1-464">**How do you calculate key influencers for numeric analysis?**</span></span>

<span data-ttu-id="53aa1-465">ในเบื้องหลัง การแสดงภาพข้อมูล AI จะใช้ [ML.NET](https://dotnet.microsoft.com/apps/machinelearning-ai/ml-dotnet) เพื่อประมวลผลการถดถอยเชิงเส้นเพื่อคำนวณผู้มีอิทธิพลหลัก</span><span class="sxs-lookup"><span data-stu-id="53aa1-465">Behind the scenes, the AI visualization uses [ML.NET](https://dotnet.microsoft.com/apps/machinelearning-ai/ml-dotnet) to run a linear regression to calculate the key influencers.</span></span> <span data-ttu-id="53aa1-466">การถดถอยเชิงเส้นเป็นแบบจำลองเชิงสถิติที่พิจารณาว่าผลลัพธ์ของเขตข้อมูลที่คุณกำลังวิเคราะห์มีการเปลี่ยนแปลงตามปัจจัยการอธิบายของคุณอย่างไร</span><span class="sxs-lookup"><span data-stu-id="53aa1-466">A linear regression is a statistical model that looks at how the outcome of the field you are analyzing changes based on your explanatory factors.</span></span>

<span data-ttu-id="53aa1-467">ตัวอย่างเช่น ถ้าเรากำลังวิเคราะห์ราคาบ้าน การถดถอยเชิงเส้นจะพิจารณาถึงอิทธิพลจากการมีห้องครัวที่ดีเยี่ยมจะมีผลต่อราคาบ้าน</span><span class="sxs-lookup"><span data-stu-id="53aa1-467">For example, if we are analyzing house prices, a linear regression will look at the impact having an excellent kitchen will have on the house price.</span></span> <span data-ttu-id="53aa1-468">บ้านที่มีห้องครัวที่ดีเยี่ยมมักมีราคาบ้านที่ต่ำกว่าหรือสูงกว่าเมื่อเทียบกับบ้านที่ไม่มีห้องครัวที่ดีเยี่ยมใช่หรือไม่</span><span class="sxs-lookup"><span data-stu-id="53aa1-468">Do houses with excellent kitchens generally have lower or higher house prices compared to houses without excellent kitchens?</span></span>

<span data-ttu-id="53aa1-469">การถดถอยเชิงเส้นยังพิจารณาจำนวนของจุดข้อมูลด้วย</span><span class="sxs-lookup"><span data-stu-id="53aa1-469">The linear regression also considers the number of data points.</span></span> <span data-ttu-id="53aa1-470">ตัวอย่างเช่น หากบ้านที่มีสนามเทนนิสมีราคาสูงกว่า แต่เรามีบ้านน้อยมากที่มีสนามเทนนิส ปัจจัยนี้จะไม่ถือว่ามีอิทธิพล</span><span class="sxs-lookup"><span data-stu-id="53aa1-470">For example, if houses with tennis courts have higher prices but we  have very few houses that have a tennis court, this factor is not considered influential.</span></span> <span data-ttu-id="53aa1-471">การกำหนดนี้เกิดขึ้นเนื่องจากมีจุดข้อมูลไม่เพียงพอที่จะอนุมานรูปแบบ</span><span class="sxs-lookup"><span data-stu-id="53aa1-471">This determination is made because there aren't enough data points available to infer a pattern.</span></span> <span data-ttu-id="53aa1-472">การทดสอบทางสถิติหรือที่รู้จักกันในชื่อการทดสอบ Wald นั้นใช้เพื่อพิจารณาว่าปัจจัยใดที่ถือว่าเป็นผู้มีอิทธิพล</span><span class="sxs-lookup"><span data-stu-id="53aa1-472">A statistical test, known as a Wald test, is used to determine whether a factor is considered an influencer.</span></span> <span data-ttu-id="53aa1-473">ภาพวิชวลใช้ค่า p เป็น 0.05 เพื่อกำหนดเกณฑ์</span><span class="sxs-lookup"><span data-stu-id="53aa1-473">The visual uses a p-value of 0.05 to determine the threshold.</span></span>

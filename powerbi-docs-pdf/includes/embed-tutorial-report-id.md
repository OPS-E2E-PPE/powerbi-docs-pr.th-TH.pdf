หากต้องการรับ GUID ID รายงานให้ทำตามขั้นตอนเหล่านี้:

1. ลงชื่อเข้าใช้บริการ Power BI

2. เปิดรายงานที่คุณต้องการฝัง

3. คัดลอก GUID จาก URL GUID คือตัวเลขระหว่าง **/reports/** และ **/ReportSection**

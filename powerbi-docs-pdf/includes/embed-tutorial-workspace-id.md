หากต้องการรับ GUID สำหรับ ID พื้นที่ทำงาน ให้ทำตามขั้นตอนเหล่านี้:

1. ลงชื่อเข้าใช้บริการ Power BI

2. เปิดรายงานที่คุณต้องการฝัง

3. คัดลอก GUID จาก URL GUID คือตัวเลขระหว่าง **/groups/** และ **/reports/**

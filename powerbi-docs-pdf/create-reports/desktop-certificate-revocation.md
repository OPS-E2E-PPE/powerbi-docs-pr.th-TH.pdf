# <a name="certificate-revocation-in-power-bi-desktop"></a>การเพิกถอนใบรับรองใน Power BI Desktop

Power BI มีสองวิธีในการเปิดใช้งานหรือปิดใช้งานการตรวจสอบใบรับรอง ในเดือนพฤศจิกายน๒๐๒๐เราได้แนะนำการตรวจสอบการเพิกถอนใบรับรองสำหรับการเชื่อมต่อเว็บใน Power BI Desktop ในตอนนี้คุณสามารถควบคุมการตรวจสอบการเพิกถอนใบรับรองได้มากขึ้น

## <a name="simple-certificate-revocation"></a>การเพิกถอนใบรับรองอย่างง่าย

ในอินเทอร์เฟซผู้ใช้ใน Power BI Desktop คุณสามารถเปิดใช้งานหรือปิดใช้งานคุณลักษณะได้

- บนเมนู **ไฟล์**>**ตัวเลือกและ**  >  **ตัวเลือก** การตั้งค่าให้เลือก **ความปลอดภัย** จากนั้นเลือกหรือยกเลิกการ **เปิดใช้งานการตรวจสอบการเพิกถอนใบรับรอง**

## <a name="more-fine-grained-control"></a>การควบคุมการปรับแต่งเนื้อเรื่องมากขึ้น

คุณสามารถควบคุมการยกเลิกการเพิกถอนใบรับรองได้มากขึ้นด้วยตัวเลือกที่สาม 

- **Basic**  การตรวจสอบพื้นฐานยอมรับใบรับรองที่ไม่รู้จักสถานะการยกเลิกตัวอย่างเช่นเนื่องจากใบรับรองไม่ได้ระบุไว้ การตรวจสอบนี้เป็นสิ่งสำคัญสำหรับองค์กรที่ใช้พร็อกซีเซิร์ฟเวอร์ของบริษัท ซึ่งเหมือนกับการเลือกกล่องกาเครื่องหมายใน Power BI Desktop
- **ปิดใช้งาน** เช่นเดียวกับการปิดใช้งานกล่องกาเครื่องหมายใน Power BI Desktop
- **ครอบคลุม** คุณสามารถปิดใช้งานหรือเปิดใช้งานการตรวจสอบการเพิกถอนในโหมดที่ *ครอบคลุม* ซึ่งไม่ยอมรับใบรับรองที่มีสถานะการยกเลิกที่ไม่รู้จัก 


|สถานะข้อมูลการเพิกถอนใบรับรอง | การตรวจสอบที่ครอบคลุม | การตรวจสอบเบื้องต้น | ไม่มี |
|---------|---------|---------|---------|
|เพิกถอน     |  ❌  | ❌  | ✔   |
|ไม่รู้จัก  |  ❌    |  ✔   |    ✔  |
|ไม่ได้ยกเลิก  | ✔  |    ✔ |    ✔  |

กล่องกาเครื่องหมายเปิดใช้งานอย่างง่ายหรือปิดใช้งานจะยังคงอยู่ในอินเทอร์เฟซผู้ใช้ Power BI Desktop หากต้องการใช้ตัวควบคุมที่ละเอียดมากขึ้นให้ตั้งค่ารีจิสทรี `DisableCertificateRevocationCheck` DWORD ดังต่อไปนี้: คุณตั้งค่านี้ในคีย์รีจิสทรี Power BI Desktop นี่เป็นหนึ่งในสิ่งเหล่านี้โดยขึ้นอยู่กับระบบปฏิบัติการของคุณ:

```
HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\Microsoft Power BI Desktop
```

หรือ

```
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Microsoft Power BI Desktop
```

ตั้งค่ารีจิสทรีเป็นหนึ่งในค่าต่อไปนี้: 

|ค่า  |โหมด  |การกำหนดค่า  |
|---------|---------|---------|
|0     | พื้นฐาน   | มีการยอมรับใบรับรองที่มีสถานะการยกเลิกที่ไม่รู้จัก เทียบเท่ากับการเปิดใช้งานกล่องกาเครื่องหมายใน Power BI Desktop |
|1     | ถูกปิดใช้งาน  | ละเว้นการตรวจสอบการเพิกถอนทั้งหมด เทียบเท่ากับการปิดใช้งานกล่องกาเครื่องหมายใน Power BI Desktop  |
|2     | ครอบคลุม  |  ต้องไม่เลิกใบรับรอง ไม่ยอมรับใบรับรองที่มีสถานะการยกเลิกที่ไม่รู้จัก |

กำหนดค่าการตั้งค่ารีจิสทรีนี้เพื่อใช้ประโยชน์จากการควบคุมแบบละเอียดมากขึ้นในวันนี้

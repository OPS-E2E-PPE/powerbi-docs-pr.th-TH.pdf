# <a name="power-bi-premium-per-user-faq-preview"></a>คำถามที่พบบ่อยเกี่ยวกับ Power BI Premium Per User (ตัวอย่าง)

Power BI **Premium Per User** ช่วยให้องค์กรสามารถให้สิทธิการใช้งานคุณลักษณะพรีเมียมสำหรับผู้ใช้แต่ละราย Premium Per User (PPU) มีความสามารถของสิทธิการใช้งาน Power BI Pro ทั้งหมดและยังเพิ่มคุณลักษณะต่าง ๆ เช่น รายงานที่มีการแบ่งหน้า AI และความสามารถอื่น ๆ ที่มีให้เฉพาะสมาชิก Premium เท่านั้น 

บทความนี้ให้คำตอบสำหรับคำถามทั่วไปเกี่ยวกับการให้สิทธิ์ใช้งาน Premium Per User ข้อมูลทั้งหมดในบทความนี้อาจมีการเปลี่ยนแปลงและมีการปรับแต่งใหม่ เนื่องจาก Premium Per User เคลื่อนผ่านช่วงการแสดงตัวอย่างและเผยแพร่สำหรับความพร้อมใช้งานทั่วไป (GA) 

![เปิดใช้งาน Premium Per User](media/service-premium-per-user-faq/premium-per-user-faq-01a.png)

บทความนี้จัดกลุ่มคำถามและคำตอบเป็นหมวดหมู่ต่อไปนี้:
* คำถามทั่วไป 
* คำถามเกี่ยวกับการดูแลระบบ 
* คำถามเกี่ยวกับประสบการณ์ของผู้ใช้ปลายทาง 

## <a name="general-questions"></a>คำถามทั่วไป

1.  **Premium Per User (PPU) คืออะไร** 

    Premium Per User (PPU) เป็นวิธีใหม่ในการให้สิทธิการใช้งานคุณลักษณะพรีเมียมสำหรับผู้ใช้แต่ละราย ซึ่งรวมถึงความสามารถของสิทธิการใช้งาน Power BI Pro ทั้งหมด พร้อมด้วยคุณลักษณะ เช่น รายงานที่มีการแบ่งหน้า AI และความสามารถอื่น ๆ ที่มีเฉพาะใน Premium วันนี้

2.  **จะวางจำหน่ายเมื่อใด**

    การพัฒนาเป็นไปอย่างต่อเนื่อง และเราคาดว่า PPU จะไม่สามารถเผยแพร่ไปยัง GA ได้จนถึงปี 2021 การแสดงตัวอย่างแบบสาธารณะสามารถใช้ได้กับทุกองค์กรและไม่มีค่าใช้จ่ายจนกว่าจะพร้อมใช้งานทั่วไป

3.  **ฉันจำเป็นต้องมีสิทธิการใช้งาน Power BI Pro และ Premium Per User หรือไม่**

    ไม่จำเป็น คุณจะได้รับความสามารถทั้งหมดของ Power BI Pro พร้อมสิทธิ์การใช้งาน Premium Per User

4.  **ฉันจะทดลองใช้งานได้อย่างไร**

    มีประสบการณ์ทดลองใช้งานในผลิตภัณฑ์และประสบการณ์ทดลองใช้งานผ่าน Microsoft 365 ผู้ใช้ทุกคนสามารถเข้าถึงประสบการณ์การทดลองใช้งานในผลิตภัณฑ์ได้โดยเปิดใช้งาน Premium Per User สำหรับพื้นที่ทำงาน ทั้งนี้องค์กรของตนต้องไม่จำกัด PPU ด้วย คุณสามารถเปิดใช้งานประสบการณ์การทดลองใช้งานผ่าน Microsoft 365 ผ่านทางพอร์ทัลได้เหมือนกับการทดลองใช้งาน Power BI Pro ในวันนี้  
    
5.  **การทดลองใช้เริ่มต้นของฉันหมดอายุแล้วและเรายังคงอยู่ในช่วงการแสดงตัวอย่าง  ฉันจะขยาย/ต่ออายุการทดลองใช้งาน Premium Per User ระหว่างช่วงการแสดงตัวอย่างได้อย่างไร**

    สำหรับลูกค้าที่กำลังใช้รุ่นทดลองในผลิตภัณฑ์ คุณสามารถเริ่มการทดลองใช้งานใหม่ได้ตลอดเวลาหลังจากการทดลองใช้เดิมของคุณสิ้นสุดลงโดยคลิกที่ไอคอนโปรไฟล์ของคุณที่มุมขวาบนของพอร์ทัล  คุณจะเห็นตัวเลือกให้เลือกเข้าร่วมการทดลองใช้อีกครั้ง  การดำเนินการดังกล่าวจะเริ่มต้นการทดลองใช้งานอีก 60 วัน  ตัวเลือกนี้จะยังคงเป็นตัวเลือกตลอดระยะเวลาการแสดงตัวอย่าง  
    
    สำหรับลูกค้าที่กำลังใช้งานรุ่นทดลองผ่านทาง Microsoft 365 ผู้ดูแลระบบผู้เช่าของคุณอาจขยายเวลาการทดลองใช้ 30 วันเดิมของคุณออกไปอีก 30 วัน นี่คือระยะเวลาสูงสุดที่สามารถขยายการทดลองได้

6.  **คุณลักษณะใดบ้างที่มีให้ใน Premium Per User**

    ตารางเปรียบเทียบคุณลักษณะเป็นดังต่อไปนี้:    

    ![การเปรียบเทียบคุณลักษณะ Premium Per User](media/service-premium-per-user-faq/premium-per-user-faq-02.png)


7.  **องค์กรของฉันมี Power BI Premium อยู่แล้ว ตอนนี้ฉันจำเป็นต้องมีสิทธิการใช้งาน Premium Per User เพื่อเผยแพร่เนื้อหาไปยังความจุของฉันหรือไม่**
    
    องค์กรของคุณอาจเลือกที่จะผนวกความจุ Premium เข้ากับสิทธิการใช้งาน Premium Per User แต่ Premium Per User ไม่จำเป็นต้องเผยแพร่เนื้อหาไปยังความจุที่มีอยู่  

## <a name="administrative-questions"></a>คำถามเกี่ยวกับการดูแลระบบ

1.  **ฉันจะเปิดใช้งาน Premium Per User (PPU) สำหรับผู้เช่าของฉันได้อย่างไร**
    
    เมื่อมีการจัดเตรียมสิทธิการใช้งานสำหรับ PPU สำหรับผู้เช่าแล้ว คุณลักษณะ PPU จะพร้อมใช้งานในพื้นที่ทำงานใดก็ตามที่คุณเปิดใช้งาน สำหรับการแสดงตัวอย่าง เพียงแค่เปิดใช้งานคุณลักษณะ Premium Per User ในเมนูดรอปดาวน์ความจุโดยเลือกรายการ Premium Per User ตามที่แสดงในภาพต่อไปนี้

    ![เปิดใช้งาน Premium Per User](media/service-premium-per-user-faq/premium-per-user-faq-01a.png)

2.  **ทำงานได้เหมือนกับความจุแบบ Premium หรือไม่ คุณสามารถเปิด/ปิดใช้งานคุณลักษณะได้หรือไม่**

    ไม่ ไม่มีการจัดการหน่วยความจำหรือ CPU สำหรับ PPU เพียงแค่ทำงานได้เหมือนกับที่ Power BI Pro ทำได้ในวันนี้เท่านั้น ผู้ดูแลระบบผู้เช่าของคุณสามารถจัดการการตั้งค่าคุณลักษณะที่เลือกได้ แต่ไม่สามารถปิดใช้งานปริมาณงาน เช่น รายงานที่มีการแบ่งหน้าได้เช่นเดียวกับที่คุณไม่สามารถปิดแดชบอร์ดได้ในวันนี้ 

3.  **ฉันจะจัดการการตั้งค่า Premium Per User ที่เปิดเผยได้ที่ไหน**

    ตัวเลือกเมนูใหม่ในการตั้งค่าผู้เช่าสำหรับ Premium Per User ช่วยให้ผู้ดูแลระบบสามารถจัดการการตั้งค่าเหล่านั้นได้

4.  **ฉันสามารถจำกัดผู้ใช้ที่สามารถสร้างพื้นที่ทำงาน PPU ได้หรือไม่**

    ได้ ในแบบเดียวกันกับที่คุณสามารถจำกัดผู้ใช้ที่สามารถสร้างพื้นที่ทำงานได้

5.  **ผู้ดูแลระบบผู้เช่าสามารถดูว่าพื้นที่ทำงานใดถูกทำเครื่องหมายเป็น Premium Per User ได้หรือไม่**

    ได้ คุณสามารถเรียกใช้สิ่งเหล่านี้ได้ในรายการเมนูพื้นที่ทำงานบนหน้าจอผู้ดูแลระบบผู้เช่า ซึ่งจะแสดงให้คุณเห็นว่าพื้นที่ทำงานใดถูกทำเครื่องหมายเป็น Premium

6.  **ฉันสามารถย้ายพื้นที่ทำงานระหว่าง Premium Per User และความจุ Premium ได้หรือไม่**

    ใช่ สำหรับความพร้อมใช้งานทั่วไป (GA) คุณจะต้องทำการรีเฟรชชุดข้อมูลหรือกระแสข้อมูลทั้งหมดที่อยู่ในพื้นที่ทำงานเมื่อคุณย้ายกลับไปเป็นความจุแบบ Premium ข้อกำหนดนี้ป้องกันการละเมิดโดยบริษัทที่ต้องการแก้ไขกลไกการชาร์จ CPU ของ Premium Gen2

7.  **มี API ความจุสำหรับ Premium Per User หรือไม่**

    มีชุด API ที่จำกัดเพื่อให้สามารถเคลื่อนย้ายพื้นที่ทำงานได้ แต่คุณไม่สามารถทำสิ่งต่าง ๆ เช่น ปิดปริมาณงานและกิจกรรมอื่น ๆ ดังกล่าวได้  
    
7.  **โครงร่างสำคัญของบริการรองรับพื้นที่ทำงาน Premium Per User หรือไม่**

    ไม่ ขณะนี้เราไม่รองรับโครงร่างสำคัญของบริการในพื้นที่ทำงาน Premium Per User  


## <a name="end-user-experience-questions"></a>คำถามเกี่ยวกับประสบการณ์ของผู้ใช้ปลายทาง

1.  **ถ้าฉันได้ทำเครื่องหมายพื้นที่ทำงานเป็น Premium Per User ผู้ใช้รายอื่นจะทราบได้อย่างไร**
    
    มีไอคอนใหม่เพิ่มเข้ามาเพื่อแสดงว่าพื้นที่ทำงานใดเป็น Premium Per User ดังที่แสดงในภาพต่อไปนี้:

    ![ไอคอน Premium Per User](media/service-premium-per-user-faq/premium-per-user-faq-03.png)    

2.  **ใครสามารถเข้าถึงเนื้อหาในพื้นที่ทำงาน/แอป Premium Per User ได้บ้าง**

    ผู้ใช้ทุกคนต้องมีสิทธิการใช้งาน Premium Per User เพื่อดูเนื้อหาในพื้นที่ทำงาน Premium Per User ซึ่งรวมถึงสถานการณ์ที่ผู้ใช้กำลังเข้าถึงเนื้อหาผ่านจุดสิ้นสุด XMLA, วิเคราะห์ใน EXCEL, แบบจำลองแบบรวม และอื่น ๆ คุณอาจให้สิทธิ์การเข้าถึงพื้นที่ทำงานแก่ผู้ใช้ที่ยังไม่มีสิทธิการใช้งาน PPU แต่พวกเขาจะได้รับข้อความแจ้งว่าไม่สามารถเข้าถึงเนื้อหาได้ พวกเขาจะได้รับพร้อมท์แจ้งให้ใช้สิทธิการใช้งานแบบทดลองหากพวกเขามีคุณสมบัติเหมาะสม ถ้าพวกเขาไม่มีคุณสมบัติเหมาะสม พวกเขาจำเป็นต้องได้รับมอบหมายสิทธิการใช้งานจากผู้ดูแลระบบผู้เช่าของพวกเขา

3.  **เนื้อหาใดบ้างที่บุคคลที่มีสิทธิ์การใช้งานแต่ละประเภทสามารถมองเห็นได้เมื่อแชร์กับพวกเขา**

    แผนภูมิต่อไปนี้อธิบายว่าบุคคลที่มีสิทธิการใช้งาน PPU สามารถดูเนื้อหาประเภทใดได้บ้าง:

    ![แผนภูมิที่ผู้ใช้สามารถดูเนื้อหาตามประเภทสิทธิการใช้งาน](media/service-premium-per-user-faq/premium-per-user-faq-04.png)   

4.  **ฉันสามารถใช้ Premium Per User สำหรับกรณีการใช้งานแบบฝังตัวได้หรือไม่**

    Premium Per User จะทำงานในลักษณะเดียวกับกรณีการใช้งานแบบฝังตัวเมื่อคุณใช้สิทธิการใช้งาน Pro คุณสามารถฝังเนื้อหาได้และผู้ใช้แต่ละคนจะต้องมีสิทธิการใช้งาน PPU เพื่อดูเนื้อหานั้น

5.  **จะเกิดอะไรขึ้นกับพื้นที่ทำงาน PPU ของฉันเมื่อสิ้นสุดระยะเวลาการแสดงตัวอย่างหากฉันเลือกที่จะไม่ซื้อสิทธิการใช้งาน Premium Per User**

    คุณและผู้ใช้ของคุณจะยังคงสามารถเข้าถึงพื้นที่ทำงานได้ อย่างไรก็ตามคุณจะไม่สามารถใช้งานเนื้อหาที่ต้องใช้ประเภทสิทธิการใช้งานนั้นได้ คุณจำเป็นต้องย้ายพื้นที่ทำงานไปยังความจุแบบ Premium หรือเพียงแค่ปิดข้อกำหนด 

6.  **ฉันจะได้รับพื้นที่เก็บข้อมูลทั้งหมดเท่าใดเมื่อใช้ Premium Per User**

    ผู้เช่าทั้งหมดจะมีขีดจำกัดพื้นที่เก็บข้อมูล 100 TB เท่ากันกับความจุแบบ Premium ในแง่ของพื้นที่เก็บข้อมูล

7.  **API การส่งออกจะพร้อมใช้งานสำหรับ Premium Per User หรือไม่**

    ขณะนี้คุณสามารถใช้งานได้ในรายงานที่มีการแบ่งหน้า และจำกัด การเรียกเพียงครั้งเดียวทุก ๆ 5 นาที  ไม่รองรับรายงาน Power BI ในขณะนี้  

8.  **การสมัครรับข้อมูลผ่านอีเมลใน PPU จะเป็นอย่างไร**

    ทุกคนที่มีสิทธิการใช้งาน PPU หรือสิทธิการใช้งาน Pro สามารถรับการสมัครสมาชิกและไฟล์แนบใดก็ตามที่มีให้ หากไฟล์แนบนั้นเหมือนกันสำหรับผู้ใช้ทุกคน ผู้ใช้ Pro จะไม่สามารถดูเนื้อหาในพอร์ทัลผลิตภัณฑ์ได้ หากมีการนำความสามารถในการสมัครสมาชิกเพิ่มเติมที่อนุญาตให้มีมุมมองข้อมูลที่แตกต่างกันสำหรับผู้รับที่แตกต่างกันจะต้องมีสิทธิการใช้งาน PPU (หรือความจุแบบ Premium) เพื่อใช้ความสามารถเหล่านั้น

9.  **ฉันสามารถรีเฟรชเกิน 48 ครั้งผ่านพอร์ทัลโดยใช้ API การรีเฟรชได้หรือไม่**

    ไม่จำกัดการรีเฟรชแบบนี้ในขณะนี้  

10. **ฉันสามารถใช้แอปเมตริก Power BI Premium Capacity เพื่อตรวจสอบการใช้งาน Premium Per User ได้หรือไม่**

    แอปเมตริก Power BI Premium Capacity ใช้ไม่ได้กับ Premium Gen2 ดังนั้นจึงไม่อนุญาตให้คุณดูรายการ PPU ใดก็ตาม  

11. **ฉันสามารถโฮสต์ชุดข้อมูล Power BI ไปยังพื้นที่ทำงาน PPU สร้างรายงานเทียบกับพื้นที่ทำงาน เผยแพร่ในพื้นที่ทำงานที่ไม่ใช่ PPU และอนุญาตให้ผู้ใช้เข้าถึงรายงานนี้โดยไม่ต้องมีสิทธิการใช้งาน PPU ได้หรือไม่**

    หมายเลข เนื่องจากชุดข้อมูลอยู่ในพื้นที่ทำงานแบบ Premium Per User จึงไม่สามารถดูได้สำหรับผู้ใช้ที่ไม่มีสิทธิการใช้งาน แม้ว่าพวกเขาจะสามารถเข้าถึงรายงานในพื้นที่ทำงานที่ไม่ใช่ PPU ได้ก็ตาม

12. **ฉันสามารถรีเฟรชแบบจำลองข้อมูล 10 GB หลายแบบจำลองพร้อมกันได้หรือไม่**

    คุณอยู่ภายใต้ข้อจำกัดเดียวกันกับที่กำหนดไว้สำหรับการรีเฟรชแบบขนานใน Premium Gen2

13. **ฉันสามารถแบ่งปันเนื้อหาที่โฮสต์ใน Premium Per User ผ่านการเผยแพร่ไปยังเว็บได้หรือไม่**

    เผยแพร่ไปยังเว็บจะทำงานในลักษณะเดียวกับเนื้อหาที่โฮสต์ในความจุแบบ Premium ในปัจจุบัน

14. **ฉันสามารถให้กระแสข้อมูลทำงานในพื้นที่ทำงานแบบ Premium Per User ซึ่งได้มีการนำเข้าสู่ชุดข้อมูล Power BI ในพื้นที่ทำงานอื่น และผู้ใช้ที่ใช้เนื้อหานั้นไม่จำเป็นต้องมีสิทธิการใช้งาน Premium Per User ได้หรือไม่**

    ตราบใดที่ผู้เขียนรายงาน Power BI มีสิทธิการใช้งาน Premium Per User สิ่งนี้จะเป็นไปได้ในช่วงระยะเวลาการแสดงตัวอย่างสำหรับ PPU

15. **ฉันสามารถใช้แอป Power BI สำหรับอุปกรณ์เคลื่อนที่กับ Premium Per User ได้หรือไม่**

    ได้ แอป Power BI สำหรับอุปกรณ์เคลื่อนที่ได้รับการอัปเดตเพื่อทำงานกับเนื้อหาใดก็ตามที่เผยแพร่ไปยังแอปหรือพื้นที่ทำงานแบบ Premium Per User

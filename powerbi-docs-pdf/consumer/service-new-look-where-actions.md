# <a name="the-new-look-where-did-the-actions-go"></a>'รูปลักษณ์ใหม่ ' การดำเนินการไปที่ไหน

บริการของ Power BI (app.powerbi.com) ได้ย้ายไปยังรูปลักษณ์ใหม่ที่ทำให้ง่ายต่อการดูและโต้ตอบกับรายงานของคุณ ดังนั้น จึงมีการย้ายเมนูการดำเนินการหลายเมนู ถ้าองค์กรของคุณได้ทำการย้ายไปยังรูปลักษณ์ใหม่แต่คุณคุ้นเคยกับลักษณะเก่าบทความนี้จะช่วยให้คุณค้นหาวิธีการของคุณ ตารางในบทความนี้ทำการแม็ปตำแหน่งที่ตั้งเก่าไปยังตำแหน่งที่ตั้งใหม่ 

ต้องการเกี่ยวกับรูปลักษณ์ใหม่ใน **Power BI Desktop** อย่างนั้นหรือ คุณสามารถอ่านได้ที่ใช้ [ชุดแถบเครื่องมือที่อัปเดตแล้วใน Power BI Desktop](../create-reports/desktop-ribbon.md)
